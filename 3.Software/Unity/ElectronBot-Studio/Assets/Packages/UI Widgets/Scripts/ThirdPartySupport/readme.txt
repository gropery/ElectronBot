1. Data Bind for Unity support
	You can enable Data Bind for Unity support with Project Settings / New UI Widgets / Enable DataBind support.
	If Data Bind for Unity not installed option will not be available.

2. TextMeshPro support
	You can enable TextMeshPro support with Project Settings / New UI Widgets / Enable TextMeshPro support.
	If TextMeshPro not installed option will not be available.
